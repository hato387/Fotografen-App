# Design System — Naturnah

> Ruhiges, naturnahes Theme für die Naturfotografie-App. Lesbarkeit vor Effekt.

## Prinzipien
- Ruhig, viel Weißraum, abgerundete Karten
- Lesbarkeit der vielen Beschreibungstexte (Motive, Fototipps) steht im Vordergrund
- Gedeckte, naturnahe Töne — kein grelles UI
- Optionaler Dark Mode (praktisch bei Dämmerung/früh morgens im Feld)

## Typografie
- **Schrift:** Inter
- Klar, sachlich, sehr gut lesbar für lange Beschreibungstexte

## Komponenten
- Basis: Tailwind CSS + shadcn/ui
- Abgerundete Ecken, dezente Schatten
- Karten-basiertes Layout für Motive, Spots, Journal-Einträge
